# Almacén Natural - Sitio Web de Productos Alimenticios

## Propósito del sitio

Este sitio web está diseñado para **Almacén Natural**, una tienda especializada en la venta de productos alimenticios saludables y orgánicos. El objetivo principal es:

- Mostrar el catálogo de productos disponibles para la venta
- Permitir a los clientes conocer la calidad y variedad de los productos
- Facilitar el contacto con la tienda a través de un formulario
- Compartir reseñas de clientes satisfechos
- Proporcionar información de contacto y ubicación

## Tecnologías utilizadas

- **HTML5**: Estructura semántica del sitio
- **CSS3**: Estilos, diseño responsivo y animaciones
- **Flexbox**: Organización de la sección de productos
- **CSS Grid**: Organización de la sección de reseñas
- **Media Queries**: Adaptación a diferentes tamaños de pantalla
- **Formspree**: Manejo del formulario de contacto

## Características principales

✅ Diseño responsivo que se adapta a dispositivos móviles, tablets y desktop  
✅ Navegación intuitiva con menú sticky  
✅ Sección de productos con cards organizadas con Flexbox  
✅ Sección de reseñas con Grid CSS  
✅ Formulario de contacto funcional con Formspree  
✅ Contenido multimedia (video de YouTube)  
✅ Fuentes de Google Fonts (Poppins y Playfair Display)  
✅ Gradientes y efectos visuales modernos  

## Instalación y uso

1. Clona este repositorio
2. Abre el archivo `index.html` en tu navegador
3. Para el formulario de contacto, reemplaza `tu-id-de-formspree` en el action del formulario con tu endpoint de Formspree

## Configuración de Formspree

Para que el formulario de contacto funcione correctamente:

1. Regístrate en [Formspree](https://formspree.io/)
2. Crea un nuevo proyecto
3. Copia el endpoint que te proporcionan (se ve como `https://formspree.io/f/xxxxxx`)
4. Reemplaza en el archivo HTML la línea: `action="https://formspree.io/f/tu-id-de-formspree"`
